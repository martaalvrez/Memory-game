WHAT IS MEMORY?

Memory is a cognitive process that allows us to encode, store, and retrieve information.

There are three types of memory:

  • Short-term memory: Allows retaining information for a few seconds or minutes. For example, remembering a number while dialing it.
  • Working memory: It is the active form of short-term memory, and it also manipulates information. For example, remembering three numbers and performing operations with them.
  • Long-term memory: Requires repetition and consolidation. There are several types:
    • Episodic: personal memories.
    • Semantic: general knowledge, vocabulary.
    • Procedural: habits and skills, such as riding a bicycle.

WHAT ARE THE CAUSES OF MEMORY LOSS?

Memory loss, in many cases, is a natural part of aging. It causes small forgetfulness and slowy er learning of new things, but it does not lead to drastic memory loss.

However, in other cases, memory loss is due to pathological factors, such as neurodegenerative diseases (e.g., Alzheimer’s) or brain injuries. In these cases, memory loss cannot always be prevented, but it can be slowed down by stimulating and exercising memory.

Another very important factor that affects memory is lifestyle. Stress, lack of sleep, and alcohol consumption negatively impact the ability to remember and can worsen memory loss.

Thus, memory loss can be transient or progressive

PROTECTIVE FACTORS AGAINST MEMORY LOSS

There are some factors that can help slow down memory loss and your mind active:

  • Regular physical exercise
  • Healthy diet
  • Adequate sleep
  • Stress reduction
  • Cognitive stimulation: reading, learning languages, solving problems, and memory games keep the brain active.
